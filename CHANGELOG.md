# Changelog

All notable changes to the **Utilitybend Neon** theme will be documented here.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-19

### Added
- Initial public release of Utilitybend Neon.
- Full editor, workbench, terminal, peek-view, diff, and Git decoration colors.
- TextMate `tokenColors` covering HTML, CSS/SCSS, JS/TS, JSON, YAML, and Markdown.
- Semantic highlighting token colors for TypeScript/JavaScript.
